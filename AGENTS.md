# PROJECT KNOWLEDGE BASE

SvelteKit static site that generates a printable resume from MDX content.
Stack: TypeScript, Vite, Svelte, Tailwind, MDSveX, Paraglide i18n.
Deploys to GitHub Pages.

## CONVENTIONS

- **Package manager**: pnpm
- **Path aliases**: `$components` → `./src/components`, `$layouts` → `./src/layouts`, `$lib` → `./src/lib`

## ANTI-PATTERNS

- **Never edit `src/lib/translations/`** — auto-generated by Paraglide Vite plugin. Edit source files in `translations/messages/` instead
- **No SSR/server routes** — static adapter with `fallback: null`. Adding `+server.ts` or `+page.server.ts` will break the build

## COMMANDS

```bash
pnpm dev          # Vite dev server
pnpm build        # Static build → build/
pnpm preview      # Preview built site
pnpm check        # svelte-kit sync + svelte-check (type checking)
pnpm check:watch  # Type checking in watch mode
pnpm lint         # Prettier check + ESLint
pnpm format       # Prettier write
```
