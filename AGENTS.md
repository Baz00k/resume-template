## OVERVIEW

SvelteKit 2 static site that generates a printable resume from MDX content. Stack: Svelte 4, Vite 5, TypeScript 5, Tailwind 3, MDSveX, Paraglide i18n. Deploys to GitHub Pages.

## STRUCTURE

```
.
├── src/
│   ├── components/        # 4 Svelte components (Header, ExportToPdf, LanguageSwitcher, PageBreak)
│   ├── layouts/            # Resume templates selected via MDX frontmatter
│   │   ├── basic/          # Default — plain prose wrapper
│   │   └── tokyo/          # Tokyo Night themed — custom fonts + colors
│   ├── lib/
│   │   ├── translations/   # AUTO-GENERATED by Paraglide — do not edit
```

## CONVENTIONS

- **Package manager**: pnpm 9 (enforced via `packageManager` field)
- **Formatting**: Tabs, single quotes, no trailing commas, 100 char width (`.prettierrc`)
- **Path aliases**: `$components` → `./src/components`, `$layouts` → `./src/layouts`, `$lib` → `./src/lib`
- **File extensions**: `.svelte`, `.svelte.md`, `.md`, `.mdx`, `.svx` all valid
- **Static only**: `prerender = true` at root layout — no SSR, no server endpoints
- **TypeScript**: Strict via `@total-typescript/tsconfig`, bundler module resolution
- **ESLint**: Flat config format (ESLint 9)
- **Icons**: `@iconify/svelte` — use `<Icon icon="set:name" />`, global `.iconify` class in `app.css`

## ANTI-PATTERNS (THIS PROJECT)

- **Never edit `src/lib/translations/`** — auto-generated by Paraglide Vite plugin. Edit source files in `translations/messages/` instead
- **No SSR/server routes** — static adapter with `fallback: null`. Adding `+server.ts` or `+page.server.ts` will break the build
- **PDF export is client-only** — relies on `window.print()` and DOM measurement. Will not work in Safari or on mobile (button hidden on mobile via CSS `hidden md:flex`)

## UNIQUE STYLES

- **MDX layout selection**: Resume templates chosen via frontmatter `layout: basic|tokyo`, resolved in `mdsvex.config.js` — not via SvelteKit `+layout.svelte` nesting
- **Resume container**: `#resume` div in `+layout.svelte` is A4 sized (`min-h-[29.7cm] max-w-[21cm]`) with print-specific absolute positioning
- **Dynamic @page sizing**: `exportToPDF.ts` measures actual element dimensions and creates a custom `@page` size to print as single page (non-Safari)
- **Tokyo Night typography**: Full custom prose theme in `tailwind.config.cjs` with light/dark variants matching Tokyo Night VS Code color scheme
- **Smartypants**: MDSveX converts `--` to em-dashes, `...` to ellipses, quotes to curly quotes
